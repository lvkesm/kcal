Gépház

Az API kulcsodat biztonságosan, kizárólag a telefonod helyi tárhelyén (localStorage) tároljuk. Soha nem küldjük el harmadik félnek vagy külső szerverre.

Hogy szerezhetsz kulcsot?

  1. Látogass el a Google AI Studio oldalra.
  2. Kattints a "Get API Key" gombra.
  3. Hozz létre egy új kulcsot teljesen ingyen.
  4. Másold ki, illeszd be ide felülre, and mentsd el!
